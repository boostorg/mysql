ssl_options::ssl_options

Default and initialization constructor.

Synopsis 
ssl_options(
    ssl_mode mode = ssl_mode::enable);
Description

By default, SSL is enabled for the connection if the server supports it (ssl_mode::enable). See ssl_mode.
