resultset::async_read_all (1 of 4 overloads)

Reads all remaining rows.

Synopsis 
template<
    class CompletionToken>
auto
async_read_all(
    rows& output,
    CompletionToken&& token);
Description

The contents of output will be replaced by the read rows, so you can obtain the number of read rows using output.size(). After this operation succeeds, this->complete() == true.

This function requires fitting all the rows in the resultset into memory. It is a good choice for small resultsets. The handler signature for this operation is void(boost::mysql::error_code).
