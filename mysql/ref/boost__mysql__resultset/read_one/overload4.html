resultset::read_one (4 of 4 overloads)

Reads a single row as a row_view.

Synopsis 
row_view
read_one(
    use_views_t);
Description

If a row was read successfully, returns a non-empty row_view. If there were no more rows to read, returns an empty row_view.

The returned view points into the connection that *this references. It will be valid until the connection performs any other read operation or is destroyed.
