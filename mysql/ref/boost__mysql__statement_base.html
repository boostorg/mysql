class statement_base
Member Functions

Name

Description

id

Returns a server-side identifier for the statement (unique in a per-connection basis).

num_params

Returns the number of parameters that should be provided when executing the statement.

valid

Returns true if the object represents an actual server statement.

Description

Don't instantiate this class directly - use statement instead.

All member functions, except otherwise noted, have this->valid() as precondition. Calling any function on an invalid statement results in undefined behavior.
