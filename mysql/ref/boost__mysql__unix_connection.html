Extends connection with additional functions that require the undrlying stream to be a socket. In general, prefer this class over connection. See also tcp_ssl_connection, unix_ssl_connection, tcp_connection and unix_connection for the most common instantiations of this template class. See this section for more info. The same considerations for copy and move operations as for connection apply.
