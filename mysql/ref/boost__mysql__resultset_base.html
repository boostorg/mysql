resultset_base

The base class for resultsets.

Defined in header <boost/mysql/resultset_base.hpp>

Member Functions

Name

Description

affected_rows

The number of rows affected by the SQL statement that generated this resultset.

complete

Returns whether the resultset has been completely read or not.

info

Additionat text information about the execution of the SQL statement that generated this resultset.

The last insert ID produced by the SQL statement that generated this resultset.

meta

Returns metadata about the columns in the query.

valid

Returns true if the object represents an actual resultset.

warning_count

The number of warnings produced by the SQL statement that generated this resultset.

Description

Don't instantiate this class directly - use resultset instead.

All member functions, except otherwise noted, have this->valid() as precondition. Calling any function on an invalid resultset results in undefined behavior.
